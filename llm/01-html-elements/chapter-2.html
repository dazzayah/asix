Capítulo 2: Etiquetas, atributos, elementos y sintaxis HTML

2.2. El primer documento HTML

El lenguaje HTML es tan sencillo que practicamente se entiende sin estudiar el significado de sus etiquetas principales.

Además de textos en negrita, también se pueden poner en cursiva o tachado.

2.3. Etiquetas y atributos

Como indica la documentación, existen los enlaces mediante anclas.

Para que estas funcionen correctamente, necesitan ciertos parámetros como los que le he puesto en este caso para el ancla, que apuntan hacia un recurso. A estos se les llaman atributos.

Sin ellos, las etiquetas se comportan de forma predeterminada, y en algunos casos, no hacen nada.

2.4. Elementos HTML

Además de etiquetas y atributos, HTML define el término elemento para referirse a las partes que componen los documentos HTML.

El lenguaje HTML clasifica a todos los elementos en dos grupos: elementos en línea y elementos de bloque.
La diferencia entre estos está en el espacio que ocupan en la página, siendo el elemento en bloque el que siempre empieza en una línea nueva y ocupando todo el espacio hasta el final de la línea. Por otro lado el elemento en línea tan solo coge el espacio necesario para mostrar su contenido.

Los párrafos son elementos de bloque.

Los enlaces son elementos en línea

Dentro de un párrafo, los enlaces siguen siendo elementos en línea.
