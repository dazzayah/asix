## Comandos iniciales
### SHOW

Utilizado sobre los distintos elementos del GBD, como:
* Las bases de datos (DATABASES)
* Las tablas (TABLES)

````
SHOW DATABASES;
````

### USE

Utilizado para seleccionar una base de datos sobre la que trabajar:

````
USE DB_NAME;
````
